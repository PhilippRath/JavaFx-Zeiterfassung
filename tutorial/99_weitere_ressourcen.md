# Kapitel 99: Weiterführende Ressourcen & Links (Deutsch)

Um die gelernten Konzepte noch weiter zu vertiefen oder bei Problemen nachzuschlagen, findest du hier eine kuratierte Liste von deutschsprachigen Webseiten und YouTube-Ressourcen.

---

## 💻 1. JavaFX & GUI-Entwicklung

### Web-Anleitungen & Blogs
*   **[Falconbyte.net JavaFX Tutorials](https://falconbyte.net/)**
    *   *Beschreibung:* Eine der besten deutschsprachigen Schritt-für-Schritt-Anleitungen für JavaFX. Zeigt verständlich, wie man JavaFX ab Java 11/17 (modularisiert mit Maven) aufsetzt.
*   **[Entwickler.de: JavaFX Einstieg](https://entwickler.de/)**
    *   *Beschreibung:* Fachartikel rund um das JavaFX-Ökosystem und die Entwicklung von Desktop-UIs.

### YouTube-Videos & Playlists
*   **[YouTube-Kanal: ITcademy (JavaFX Playlist)](https://www.youtube.com/results?search_query=ITcademy+JavaFX+Deutsch)**
    *   *Inhalt:* Sehr strukturierte und anfängerfreundliche Videoreihe auf Deutsch. Behandelt Layout-Container, FXML-Einbindung und grundlegende Event-Steuerung.
*   **[YouTube-Kanal: codingenieur (JavaFX Reihe)](https://www.youtube.com/results?search_query=codingenieur+JavaFX+Deutsch)**
    *   *Inhalt:* Gut verständliche Erklärungen zu Stages, Scenes, dem Scene Builder und der MVC-Struktur.

## 🏛️ 2. Software-Architektur (MVC & DAO-Pattern)

### Web-Artikel
*   **[Wikipedia: Data Access Object (DAO)](https://de.wikipedia.org/wiki/Data_Access_Object)**
    *   *Beschreibung:* Kompakte, theoretische Einführung in das DAO-Pattern und die Kapselung von Datenbankzugriffen.
*   **[Centron Blog: Das DAO-Muster verstehen](https://www.centron.de/)**
    *   *Beschreibung:* Anschauliche Erklärung zur Trennung von Datenzugriff und Logik in Enterprise-Anwendungen.

### YouTube-Videos
*   **[YouTube-Suche: "DAO Pattern Java Deutsch"](https://www.youtube.com/results?search_query=DAO+Pattern+Java+Deutsch)**
    *   *Inhalt:* Kurze Erklärvideos, die das DAO-Prinzip anhand von einfachen Codebeispielen veranschaulichen.

## 🗃️ 3. Datenbanken (JDBC & SQLite)

### Web-Ressourcen
*   **[Baeldung: JDBC Basics (Englisch, aber Standardwerk)](https://www.baeldung.com/java-jdbc)**
    *   *Beschreibung:* Auch wenn es Englisch ist: Die beste, fehlerfreie Übersicht über JDBC-Verbindungen, `PreparedStatements` und ResultSets.
*   **[YouTube-Suche: "SQLite in Java einbinden Deutsch"](https://www.youtube.com/results?search_query=Java+SQLite+JDBC+Deutsch)**
    *   *Inhalt:* Praktische Video-Guides zur Einbindung der dateibasierten SQLite-Datenbank in Java-Projekte.
